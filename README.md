# nsideairbnb
InsideAirbnb  InsideAirbnb es un proyecto de análisis y visualización de datos que tiene como objetivo explorar, entender y transparentar el impacto real de Airbnb en ciudades de todo el mundo. A través de datos abiertos, el proyecto permite estudiar cómo el alquiler a corto plazo afecta la vivienda, el turismo y las comunidades locales.
